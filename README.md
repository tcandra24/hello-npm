# hello-npm-tito
A Simple Hello World Example Package

## Installation
Install via NPM:

```bash
npm install hello-npm-tito

```

## Usage

#### javascript

```javascript

var myApp = require("hello-npm-tito");

myApp.printMsg(); // print and return " Hello World! " 

```

#### TypeScript

```typescript

import * as myApp from 'hello-npm-tito';

myApp.printMsg(); // print and return " Hello World! " 


```
